# 📊 Marketing Funnel & Conversion Analysis

Project Overview

The goal is to identify:
* Where users drop off in the funnel
* Which segments generate high-quality leads
* How conversion performance can be improved

Tools Used
* Python (data cleaning & preprocessing)
* Power BI (dashboard & visualization)

Key Metrics
* **Total Contacts:** 41,000+
* **Total Conversions:** 5,000+
* **Conversion Rate:** 11.27%

 Key Insights

1. Customer Segmentation (Job)
* Students and retired clients show the highest conversion rates.
* Administrative and blue-collar segments underperform in comparison.
Not all leads are equal — targeting high-performing segments can significantly improve ROI.

2. Channel Effectiveness (Contact Type)
* Cellular communication significantly outperforms telephone contact.
Customers respond better to direct/mobile communication channels.

3. Seasonal Trends (Month)
* Conversion rates peak in March, December, and September.
* Lower performance observed in mid-year months.
 Customer behavior is seasonal — timing plays a key role in campaign success.

4. Campaign Effectiveness (Contact Frequency)
* Conversion rates decrease as campaign intensity increases.
* “Low” contact groups outperform “High” and “Very High”.
Over-contacting leads to diminishing returns and possible customer fatigue.

Recommendations
* Focus marketing efforts on high-converting segments (students, retired, etc.)
* Prioritize cellular communication channels
* Optimize campaign timing based on high-performing months
* Reduce excessive contact attempts to avoid customer fatigue
* Improve targeting to increase lead quality instead of volume

Conclusion
This analysis highlights how data-driven decisions can improve marketing efficiency and directly impact revenue by optimizing targeting, timing, and communication strategy.
